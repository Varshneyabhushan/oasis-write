Oasis Write

A minimal, distraction-free markdown editor designed for writers who value clarity and focus.

What is Oasis Write?

Oasis Write is a native desktop application built with Tauri and React that provides a serene space for your markdown writing. No complex menus, no overwhelming toolbars—just you and your words.

"The best writing tool is the one that gets out of your way."

Features

Live Markdown Rendering

See your markdown come to life as you type. Format text using bold, italic, and inline code without switching between editor and preview modes.

// Write code blocks with proper formatting
function greet(name) {
  return `Hello, ${name}!`;
}

Document Organization

  • File tree for managing your markdown documents
  • Automatic outline generation from headings
  • Quick navigation between sections
  • Auto-save with status indicators

Distraction-Free Writing

Toggle the sidebar to enter zen mode. Hide everything except your words. Focus on what matters.

Why Choose Oasis Write?

  1. Fast & Lightweight: Built with Tauri and Rust for quick startup and minimal resource usage
  2. Privacy First: Everything stays local. No cloud sync, no tracking, no accounts required
  3. Open Source: Free forever. Inspect the code, contribute, or fork it for your own use
  4. Cross-Platform: Works seamlessly on macOS, Windows, and Linux

Get Started

Ready to create your writing oasis? Download the app for your platform and open a folder containing markdown files. That's it.
